    class Market

    People exchange commodities in markets. Each market focuses on certain commodity pair `{A, B}`. By convention, we call people exchange A for B sellers who submit ask orders, and people exchange B for A buyers who submit bid orders.

    ID of market is always in the form “#{B}#{A}”. For example, in 'btccny' market, the commodity pair is `{btc, cny}`. Sellers sell out btc for cny, buyers buy in btc with cny. btc is the `base_unit`, while cny is the `quote_unit`.

def scope?(account_or_currency)
  code = if account_or_currency.is_a? Account
           account_or_currency.currency
         elsif account_or_currency.is_a? Currency
           account_or_currency.code
         else
           account_or_currency
         end

  base_unit == code || quote_unit == code
end
